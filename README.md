# Intégration Groupe 3F pour Home Assistant

Récupère la consommation d'eau (Chaude et Froide) depuis l'espace client Groupe 3F.

## Fonctionnalités
- Support de l'authentification à deux facteurs (2FA/Email/SMS).
- Découverte automatique du contrat.
- Capteurs compatibles avec le Dashboard Énergie.

## Installation via HACS
1. Ajoutez ce dépôt en tant que "Dépôt personnalisé" dans HACS.
2. Cherchez "Groupe 3F" et installez.
3. Redémarrez Home Assistant.
4. Allez dans Paramètres > Appareils et services > Ajouter > Groupe 3F.
